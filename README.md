# genAI-BookMyShowClone

A modern full-stack movie ticket booking platform inspired by the user experience of BookMyShow, developed using an AI-assisted rapid application development workflow with Lovable, React, TypeScript, Tailwind CSS, and Supabase.

The project demonstrates how generative AI tooling can accelerate frontend architecture, UI composition, and iterative product development while maintaining a scalable and production-oriented codebase structure.

## Overview

This application replicates the core functionality of a modern online movie ticket booking platform, including authentication, movie discovery, seat reservation, payment flow simulation, and booking confirmation.

The project was developed using an AI-assisted engineering workflow, where generative AI tools were extensively leveraged for UI generation, rapid prototyping, component structuring, and development acceleration.

---

## Key Features

* Responsive landing page with featured movies and dynamic poster layouts
* Secure user authentication using Supabase Auth
* Real-time movie and booking data management
* AI-assisted personalized movie recommendation workflow
* Interactive theater and showtime browsing
* Dynamic seat selection interface with live availability indicators
* Mock payment integration supporting UPI and card workflows
* Booking confirmation and reservation summary screens
* Horizontal date-scroll navigation for future show scheduling
* Fully responsive design optimized for desktop and mobile devices
* Modular and maintainable component architecture

## Local Development Setup

### 1. Clone the Repository

```bash
git clone https://github.com/<your-username>/genAI-BookMyShowClone.git
cd genAI-BookMyShowClone
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```
